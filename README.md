## Sobre o Projeto

O **Cafe Aroma e Sabor** e um sistema web full-stack para gerenciamento de estoque de cafeteria. Desenvolvido com Spring Boot e Thymeleaf, oferece uma interface elegante e intuitiva para controle de produtos, movimentacoes de estoque e usuarios do sistema.

A paleta de cores foi inspirada nos tons do cafe: marrom escuro, dourado e bege, criando uma experiencia visual aconchegante e profissional.

<br>

## Funcionalidades

- **Catalogo de Produtos** -- Cadastro, edicao, listagem e exclusao de produtos com validacao de dados
- **Controle de Estoque** -- Registro de movimentacoes (entrada/saida) com atualizacao automatica do saldo
- **Gestao de Usuarios** -- Administracao de usuarios do sistema com controle de status ativo/inativo
- **Dashboard** -- Visao geral com metricas de produtos cadastrados e movimentacoes realizadas
- **Interface Responsiva** -- Layout adaptavel para desktop e dispositivos moveis
- **Tema Personalizado** -- Design exclusivo com identidade visual da marca

## Pre-requisitos

Antes de executar o projeto, voce precisa ter instalado:

- **Java 21** ou superior
- **Maven** (ou utilize o Maven Wrapper incluso `mvnw`)
- **MySQL** (opcional, o projeto pode rodar com H2 em memoria)

<br>

## Como Executar

### 1. Clone o repositorio

```bash
git clone https://github.com/hrsonn/CafeAroma-Sabor.git
cd CafeAroma-Sabor
```

### 2. Execute com Maven

```bash
# Usando o Maven Wrapper (recomendado)
./mvnw spring-boot:run

# Ou usando Maven instalado
mvn spring-boot:run
```

### 3. Acesse a aplicacao

Abra o navegador em [http://localhost:8080](http://localhost:8080)

### 4. Login padrao

```
Usuario: admin
Senha:   senha123
```

<br>

## Configuracao MySQL (Opcional)

Por padrao, o projeto usa H2 em memoria. Para usar MySQL:

1. Crie um banco de dados MySQL chamado `cafe_aroma_sabor`
2. Edite `src/main/resources/application-mysql.properties` com suas credenciais
3. Execute com o profile MySQL:

```bash
./mvnw spring-boot:run -Dspring-boot.run.profiles=mysql
```

## Endpoints

### Paginas Web

| Rota                    | Descricao                 |
|-------------------------|---------------------------|
| `/` ou `/home`          | Dashboard com metricas    |
| `/login`                | Pagina de autenticacao    |
| `/produto`              | Listagem de produtos      |
| `/produto/cadastrar`    | Cadastro de produto       |
| `/estoque`              | Movimentacoes de estoque  |
| `/usuario`              | Listagem de usuarios      |
| `/usuario/cadastrar`    | Cadastro de usuario       |
| `/h2-console`           | Console H2 (dev apenas)   |
